🎓 University Recommendation System
📌 Project Overview

This project is a web-based University Recommendation System that suggests universities based on student academic performance.

🚀 Features
Student Login System
Graduate Recommendation (GRE + CGPA)
Undergraduate Recommendation (SAT + Fees)
University Search
University Comparison
Scholarship Suggestions
🧠 Recommendation Logic

The system uses rule-based logic inspired by:

K-Nearest Neighbors (KNN)
Feature-based filtering
📊 Dataset
File: Processed_data.csv
Contains GRE, CGPA, and University names
Used for building recommendation logic
💻 Technologies Used
HTML (Structure)
CSS (Design)
JavaScript (Logic)
LocalStorage (Login system)
🌐 Live Website

👉 https://university-recommendation-system--nishamuruganamm.replit.app/

📁 Project Structure
index.html → Homepage
login.html → User login
graduate.html → Graduate recommendation
undergraduate.html → Undergraduate recommendation
search.html → Search universities
compare.html → Compare universities
scholarships.html → Scholarship info
style.css → UI design
script.js → Logic
🎯 Conclusion

This project helps students choose universities based on their academic profile and improves decision-making using data.
